# Fregat — Школа тендерного обучения

Проект: Лендинг для онлайн-школы тендерного обучения «Фрегат»

## Project Knowledge

Вся документация проекта хранится в `.claude/skills/project-knowledge/references/`:

- `project.md` — назначение проекта, аудитория, ключевые фичи
- `architecture.md` — технический стек, структура проекта
- `patterns.md` — конвенции кода, git workflow, бизнес-правила
- `deployment.md` — платформа хостинга, переменные окружения
- `ux-guidelines.md` — UI-язык, тон общения, доменный глоссарий
- `team.md` — профили преподавателей и команды

## Репозитории

- **Основной:** [suyazov/fregat](https://github.com/suyazov/fregat) — только сайт (HTML, CSS, assets)
- **Документация:** [suyazov/fregat-docs](https://github.com/suyazov/fregat-docs) — проектная документация

## Синхронизация документации

⚠️ **ВАЖНО:** При изменениях контента о школе, команде или бизнес-правилах:

1. Сначала обновить сайт в основном репо (`fregat/`)
2. Затем предложить: *"Обновить документацию в fregat-docs?"*
3. При согласии — обновить соответствующие файлы в `fregat-docs/.claude/skills/project-knowledge/references/`
4. Закоммитить и запушить в оба репозитория

**Примеры триггеров для обновления документации:**
- Изменение данных о преподавателях → `team.md`
- Новые бизнес-правила или лицензия → `project.md`
- Изменение тарифов или сроков → `project.md`
- Новые контактные данные → `project.md`
