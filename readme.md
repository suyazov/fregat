# Проект: Лендинг для школы тендерного обучения «Фрегат»

## 1. О проекте
Продающий лендинг (Landing Page) для онлайн-школы по подготовке специалистов в сфере госзакупок (тендеров). Сайт разработан на основе анализа 5 ключевых конкурентов и существующего фирменного стиля компании.

**Цель:** Привлечение заявок на обучение профессии "Тендерный специалист" с получением диплома гос. образца.

## 2. Дизайн-концепция
**Стиль:** Премиальный, строгий, "дорогой".
**Цветовая гамма:** Глубокий темно-изумрудный фон (`#142125`) + золотые акценты.\
**Ключевые фишки:**
*   Геометрические фигуры (стрелки, ромбы) в фоне, видимые при скролле.
*   Вертикальный брендинг слева (декоративная надпись).
*   Анимации появления контента при прокрутке.
*   Прогресс-бар прокрутки сайта.

## 3. Структура сайта (Блоки)

### Hero (Первый экран)
*   УТП: Профессия "Тендерный специалист".
*   Статистика: +45% рост зарплаты, 3 мес. обучения, 44-ФЗ, 80% практика.
*   Кнопки призыва к действию (CTA).

### Блок "Практика — Теория"
*   Акцент на соотношении 80% практики и 20% теории. Закрывает возражение о "скучной теории".

### Преимущества
*   Лицензия ДПО (доверие).
*   Практика на реальных площадках.
*   Помощь в трудоустройстве.
*   Удобный онлайн-формат.

### Слайдер "Для кого обучение"
*   Сегментация аудитории: Новички, Предприниматели, Руководители.
*   Для каждого сегмента свой слайд с УТП.

### Зарплаты (Финансовая мотивация)
*   Визуализация прогрессии дохода: Junior (45-60к) -> Middle (80-120к) -> Senior (150к+).
*   Анимированные полоски прогресса.

### Как проходит обучение
*   4 шага: Тестирование -> Уроки -> Практика -> Диплом.

### Программа обучения (8 модулей)
1.  Введение в закупки.
2.  Планирование.
3.  Способы определения поставщика.
4.  Контрактация.
5.  Работа на ЭТП.
6.  Обжалование (ФАС).
7.  223-ФЗ и коммерческие закупки.
8.  Автоматизация.

### Эксперт (Сторителлинг)
*   История успеха эксперта Галины.
*   Цитата и фото.
*   Опыт работы с крупными бюджетами.

### FAQ (Частые вопросы)
*   Нужен ли опыт?
*   Какой документ выдается?
*   Можно ли совмещать с работой?

### Стоимость (Тарифы)
*   **Тендерный специалист (25 000 ₽):** Базовый курс.
*   **Тендерный эксперт (45 000 ₽):** С наставником и трудоустройством (Хит продаж).

### Форма захвата
*   Оффер: "Индивидуальная консультация в подарок".
*   Поля: Имя, Телефон, Email.

## 4. Текущий статус и план запуска
1.  Готова структура и верстка.
2.  Установлен контент-заглушка ("рыба").
3.  **План:** Встреча с партнером для утверждения контента -> Правки -> Запуск рекламы.

*   *HSE / MBA:* Акцент на лицензии и дипломе.
