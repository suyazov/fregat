# Task 4.1: Проверить адаптивность

## Description
Тестировать страницу на различных breakpoint'ах: mobile (<640px), tablet (640-1024px), desktop (>1024px).

## Acceptance Criteria
1. Mobile view:
   - Header с гамбургер-меню работает
   - Все секции читабельны
   - Карточки stacked (1 колонка)
   - Текст не обрезается
2. Tablet view:
   - 2 колонки для карточек
   - Навигация адаптивна
3. Desktop view:
   - 3 колонки для карточек
   - Отступы корректные
4. Горизонтальный скролл отсутствует

## Technical Details
**Breakpoints to test:**
- 375px (mobile small)
- 640px (mobile large / tablet start)
- 768px (tablet)
- 1024px (desktop start)
- 1280px (desktop large)

**Common issues to check:**
- `overflow-x: hidden` на body
- Images max-width: 100%
- Text wrapping on small screens
- Touch targets minimum 44px

## Definition of Done
- Проверено на всех breakpoints
- Мобильная версия работает корректно
- Tablet и desktop отображаются как ожидается
