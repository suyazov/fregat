# Task 4.2: Валидация HTML5

## Description
Проверить валидность HTML через W3C validator и исправить критические ошибки.

## Acceptance Criteria
1. HTML passes W3C validation (no critical errors)
2. Semantic tags used correctly
3. No unclosed tags
4. Attributes properly quoted
5. IDs are unique

## Technical Details
**W3C Validator:**
- Online: https://validator.w3.org/
- Or local validation tool

**Common issues to check:**
- Unclosed `<div>`, `<section>`, `<span>`
- Duplicate IDs
- Missing alt attributes on images
- Unclosed quotes
- Invalid nesting (e.g., `<p>` inside `<p>`)

**Acceptable warnings (not blocking):**
- Deprecated attributes (if inherited from index.html)
- Vendor-specific meta tags (like CSP)

## Definition of Done
- HTML валиден
- Критические ошибки исправлены
- `verify_smoke` проходит (или validator returns clean)
