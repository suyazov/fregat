# Task 5.1: Code Audit

## Description
Holistic code quality review of all feature code. Проверить чистоту кода, консистентность с дизайн-системой, лучшие практики.

## Scope
- HTML структура и семантика
- CSS классы (использование дизайн-системы)
- Код стиль (отступы, комментарии)
- DRY принципы
- Accessibility (ARIA, semantic HTML)

## Focus Areas
1. **Consistency**: Стили соответствуют `styles.css` и `DESIGN-SYSTEM.md`
2. **Semantics**: Правильное использование HTML5 тегов
3. **Maintainability**: Читаемый код, комментарии где нужно
4. **Performance**: Нет лишних дублирований, CSS inline styles

## Deliverable
Отчёт с найденными issues (если есть). Критические — нужно исправить, minor — можно отложить.

## Definition of Done
- Код проверен

- Critical issues (если есть) documented для fixes
