# Task 5.2: Security Audit

## Description
OWASP Top 10 review страницы. Проверить CSP, input validation, XSS уязвимости.

## Scope
- Content Security Policy (CSP meta tag)
- XSS vulnerabilities (user input, dynamic content)
- External resources (CDNs, fonts)
- Sensitive data exposure (placeholder данные)

## Focus Areas
1. **CSP**: Meta tag настроен корректно, 'unsafe-inline' там где нужно
2. **External scripts**: Только доверенные CDN (Tailwind, Google Fonts)
3. **Placeholder data**: Реальные данные не泄漏
4. **Links**: Все `href` атрибуты валидны, нет `javascript:`

## Deliverable
Отчёт с найденными security issues (если есть).

## Definition of Done
- Security review выполнен
- CSP проверен
- Critical vulnerabilities documented
