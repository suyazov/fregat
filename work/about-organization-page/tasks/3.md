# Task 1.3: Реализовать хлебные крошки

## Description
Добавить breadcrumbs component с путём: `Главная → Сведения об образовательной организации`, используя стили из `styles.css`.

## Acceptance Criteria
1. Хлебные крошки расположены после header, перед основным контентом
2. Путь: Главная → Сведения об образовательной организации
3. «Главная» — кликабельная ссылка на `index.html`
4. Текущая страница не кликабельна (active state)
5. Стилизация по дизайн-коду:
   - Разделитель: `/` или `→`
   - Цвет: серый для ссылок, золотой для active
   - Hover эффект на ссылке
6. Адаптивность: скрыты на mobile, видны на tablet+

## Technical Details
**CSS pattern from styles.css:**
```css
.breadcrumb {
    display: flex;
    align-items: center;
    gap: var(--spacing-sm);
    padding: var(--spacing-md) 0;
    font-size: var(--text-sm);
}

.breadcrumb a {
    color: var(--color-text-secondary);
    text-decoration: none;
    transition: color var(--duration-base);
}

.breadcrumb a:hover {
    color: var(--color-gold);
}

.breadcrumb .active {
    color: var(--color-gold);
    font-weight: 500;
}
```

## Definition of Done
- Хлебные крошки отображаются корректно
- Ссылка «Главная» ведет на index.html
- Стили соответствуют дизайн-коду
- Адаптивность проверена
