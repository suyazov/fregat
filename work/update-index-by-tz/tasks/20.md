## Description

Specifically verify mobile version loads quickly, all text is readable — приоритетная проверка.

## What to do

1. Открыть index.html в mobile браузере или dev tools mobile view
2. Проверить загрузку страницы (нет ли длительной загрузки)
3. Проверить читаемость всех текстов на мобильном
4. Проверить touch-элементы (кнопки достаточно большие)
5. Проверить навигацию на мобильном

## Acceptance Criteria

- [ ] Mobile версия загружается быстро
- [ ] Все тексты читаемы на мобильном
- [ ] Touch-элементы достаточно крупные
- [ ] Навигация работает на мобильном
- [ ] Нет горизонтального скролла
