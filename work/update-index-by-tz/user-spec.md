# User Spec: Обновить index.html по ТЗ

## Что делаем

Обновить утверждённый заказчиком файл `index.html` в соответствии с техническим заданием от 2026-03-20:
- Внести правки в 5 существующих секций
- Добавить 3 новые секции
- Удалить форму захвата заявок
- Интегрировать фото эксперта в Hero секцию как фон
- Заменить Slider на статичный контент

## Зачем

Повысить доверие к сайту — убрать ощущение «недоделанности». Сделать сайт аккуратным, без визуальных косяков, чтобы пользователю захотелось купить обучение.

## Acceptance Criteria

### Новые секции (3)
- [ ] Секция «Забронируйте место» — блок для фиксации скидки и бронирования места с бейджем скидки и датой дедлайна
- [ ] Секция «Ближайший набор» — динамическая дата начала обучения с CTA кнопкой
- [ ] Секция «Абонементы» — 3 тарифа (названия и цены взять с конкурента https://глав-тендер.рф/, блок «Нужна помощь на работе?»)

### Изменения в существующих секциях (5)
- [ ] Hero — интегрировать фото эксперта как фон (пример: moskotin.com)
- [ ] Практика — изменить текст на «80% практики на реальных площадках. Вы научитесь, а не просто послушаете»
- [ ] Трудоустройство — изменить текст на «Стажировка в реальных компаниях. Возможность трудоустройства в ООО «Фрегат» и ООО «УКЭ»»
- [ ] Зарплаты — убрать англицизмы (Junior/Middle/Senior → Начинающий/Специалист/Ведущий специалист), изменить формат на «от 50 000 ₽ + %» (процент от прибыли по тендеру)
- [ ] Как проходит обучение — удалить «Тестирование», добавить 2-й блок программы, добавить «Практические задания с проверкой и обратной связью от эксперта», заменить «Диплом» на «Удостоверение о повышении квалификации» и «Сертификат»

### FAQ
- [ ] Расширить с 3 до 14 вопросов (взять с конкурента https://niidpo.ru/seminar/specialist-v-sfere-zakupok-340-chasov и переписать под себя)

### Удаления (2)
- [ ] Удалить форму CTA (инпуты: имя, телефон, email), оставить только кнопку с текстом «Откроется квиз»
- [ ] Удалить самостоятельную секцию Expert Block (контент интегрирован в Hero)

### Структурные изменения
- [ ] Заменить Audience Slider на статичный контент (3 карточки: Новичкам, Предпринимателям, Руководителям)
- [ ] Исправить номер лицензии на `Л035‑01217‑26/03829175`
- [ ] Исправить мета-тег description на «Удостоверение о повышении квалификации и сертификат»

### Порядок секций
[ ] Соблюдать порядок: Hero → Забронируйте место → Практика → Advantages → Slider (статичный) → Зарплаты → Как проходит обучение → Программа → Абонементы → Преподаватели → Ближайший набор → FAQ → Footer

### Качество и тестирование
- [ ] Приоритет мобильная версия — быстрая загрузка на телефоне
- [ ] Проверить адаптивность на нескольких размерах экрана
- [ ] Проверить все ссылки работают
- [ ] Проверить читаемость всех блоков
- [ ] Локально открыть файл в браузере и протестировать

### Заглушки
[ ] Все значения (даты, цены, скидки, содержание тарифов, тексты FAQ) — заглушки с плейсхолдерами

## Non-Requirements

- Ленивая загрузка НЕ требуется (это Фаза 2 — WordPress)
- Все данные статичные, без динамической подгрузки
- Формы захвата заявок НЕ используются (только кнопки CTA с плейсхолдером «Откроется квиз»)

## User Flows

1. Пользователь открывает сайт на мобильном устройстве
2. Просматривает секции в указанном порядке
3. Видит аккуратный дизайн без визуальных косяков
4. Нажимает на CTA кнопку — ожидает открытие квиза (плейсхолдер)
5. Весь контент читаемый, понятный

## Edge Cases

- Очень длинные тексты в карточках — предусмотреть корректное отображение
- Очень большие изображения — предусмотреть корректное отображение на мобильных
- Очень маленький контент (заглушки) — показать «скоро добавится»
- Медленный интернет — ничего особенного, просто дольше ждёт
